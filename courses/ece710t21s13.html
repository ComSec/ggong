Course Outline

This course introduces some timely topics in communications security research. It covers security architecture and infrastructure, trust and treat models, and practical crypto schemes. Network access authentication protocols, wireless network security, radio air link protection, and security for mobility. Broadcasting and multi-cast security. Security of ubiquitous computing includes protections in mobile ad-hoc networks (MANET), vehicular ad-hoc network (VANET), wireless sensor networks (WSN), and radio frequency identification (RFID) systems. Some advanced attacks and hardware protections

Prerequisites

Main Topics

  1. Review of basics of security: information security, protection mechanisms, confidentiality, integrity and authenticity, trust and threat model, certificate authority, and practical crypto schemes (SHA, HMAC, AES, GCM, DSS, ECC).
  2. Network access authentication protocols: authentication and key agreement, AAA, and extensible authentication protocols (EAP), attacks on password-based authentication, and tunnelled attacks on EAP/TLS.
  3. Wireless network security: special aspects of wireless protection, cellular air link protection, IEEE 802.11 security solutions, forgery attacks on integrity and authentication, security for mobility, secure handover, service protection, and jamming attacks.
  4. Broadcasting and multicast security: multicast key distribution, hash chain broadcast message authentication, Merkle tree based authentication and signatures.
  5. Security of ubiquitous computing: establishment of security association, key management in MANET,location privacy in VANET, and attestation and protection for WSN (ZigBee Smart Energy).
  6. Security of RFID systems: RFID tags and communications, privacy preserving authentication, physical layer based protection, near field communication (NFC) protection, and replay attacks.
  7. Special topics: trusted platform, tamper resistance hardware, side-channel attacks, jamming attacks, and quantum cryptography.
